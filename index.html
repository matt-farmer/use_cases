Welcome to Use Cases

A repository of the raw source documents that are used to construct the HITS use cases. Documents are in Sphinx markdown format, and will be pushed to the Read The Docs rendering engine for public access.

How to Contribute

Use the GitHub client of your choice to clone the repository, edit the source of the use case documents, and commit new versions. Updated text will then be pushed forwards into the public RTD view once reviewed.
